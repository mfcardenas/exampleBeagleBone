Demo: Grove Barometer Sensor

This Grove-Barometer Sensor（BMP180) module for the Bosch BMP180 high-precision, low-power digital barometer. The BMP180 offers a pressure measuring range of 300 to 1100 hPa with an accuracy down to 0.02 hPa in advanced resolution mode. It’s based on piezo-resistive technology for high accuracy, ruggedness and long term stability. The chip only accepts 1.8V to 3.6V input voltage. However, with outer circuit added, this module becomes compatible with 3.3V and 5V. Therefore, it can be used on Arduino/Seeeduino or Seeeduino Stalker without modification.It is designed to be connected directly to a micro-controller via the I2C bus.
Please visit SeeedStudio's wiki page for more info about this sensor.


Example

import Adafruit_BMP.BMP085 as BMP085


#sensor = BMP085.BMP085(mode=BMP085.BMP085_ULTRAHIGHRES)

sensor = BMP085.BMP085()

print 'Temp = {0:0.2f} *C'.format(sensor.read_temperature())
print 'Pressure = {0:0.2f} Pa'.format(sensor.read_pressure())
print 'Altitude = {0:0.2f} m'.format(sensor.read_altitude())
print 'Sealevel Pressure = {0:0.2f} Pa'.format(sensor.read_sealevel_pressure())




Build and execute instructions

  • Connect the Grove Barometer Sensor to J4 of the board
  • Open Cloud9 IDE in your browser.
  • In the left of the browser click "Grove Demos" folder and open the "Grove_Barometer_Sensor.py" file.
  • Click "RUN" on the Cloud9 IDE and it will output the temperature pressure and altitude information by the Terminal Window.
